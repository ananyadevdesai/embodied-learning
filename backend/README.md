# Backend

This directory contains backend services for the Embodied Learning project.

## Planned Features

- Experiment data validation against schema
- API endpoints for submitting experiments
- Data storage and retrieval
- Analytics and metrics computation

## Architecture (To Be Implemented)

- REST API (Python/Flask or Node.js/Express)
- JSON schema validation
- Database for experiment metadata
- File storage for experiment data

## Getting Started

Backend implementation is in progress. For now, experiments are submitted via GitHub PRs.

## Contributing

See main `CONTRIBUTING.md` for guidelines. Backend developers should:
- Maintain schema compatibility
- Add comprehensive validation
- Document API endpoints
- Include tests
